
# Network Diagrams

This directory contains architecture diagrams for the Enterprise SOC + Cloud Homelab.

These diagrams document the evolution of the lab infrastructure.

## Diagrams Included

### Core SOC Lab (Flat Network)

Initial lab architecture consisting of:

- Active Directory domain
- Windows client workstation
- Splunk SIEM server
- VirtualBox host-only network

### Enterprise Architecture (Planned)

Future architecture including:

- pfSense firewall
- Suricata IDS
- segmented networks (DMZ, internal, SOC)
- containerized services
- AWS cloud integration

## Diagram Types

The following diagram formats are included:

- `.drawio` → editable diagram source files
- `.png` → exported images for documentation
